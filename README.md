# Messenger

Secure PC-only messenger for a small group of friends.

## Repository layout
- `server/` — Rust backend
- `client/` — Tauri + React desktop app
- `shared/` — shared protocol and types
- `deploy/` — deployment assets
- `docs/` — setup and technical notes

## Current status
The client is now a Tauri desktop app with a React UI prototype. It should be opened through the Tauri dev flow, not as a plain web site.
